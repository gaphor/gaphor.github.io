### Winget

Install Gaphor from [winget](https://github.com/microsoft/winget-pkgs/tree/master/manifests/g/gaphor/gaphor):

```bash
winget install gaphor
```

To check out the latest release notes and other details:

```bash
winget show gaphor
```
