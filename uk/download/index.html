Завантажити Gaphor

Є багато способів встановити Gaphor. Найпростіший — завантажити офіційний інсталятор для Windows або macOS. Для Linux ви можете встановити Gaphor за допомогою FlatHub. Ви також можете використовувати вбудований інструмент Python pip, якщо у вас встановлено всі необхідні залежності.

Якщо ви розробник, ви можете знайти вихідний код на GitHub.


Windows

Завантажити інсталятор

Крім того, можна завантажити портативну версію Gaphor.

Завантажити портативну версію

Вінгет

Встановіть Gaphor із winget:

winget install gaphor

Щоб переглянути останні примітки до випуску та інші відомості:

winget show gaphor

macOS

Завантажити ARM .dmg

Завантажити Intel .dmg

Gaphor вимагає macOS 10.15 або новішої версії.

Домашнє пиво

Налийте Gaphor із самоварної бочки:

brew install --cask gaphor

Linux

Flatpak

Flatpak — рекомендований спосіб встановлення Gaphor у Linux. Якщо ви ще не налаштували його, дотримуйтеся вказівок, щоб інсталювати Flatpak.

Щоб установити Gaphor вручну:

flatpak remote-add --user --if-not-exists \
    flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install --user flathub org.gaphor.Gaphor

Arch Linux

Gaphor можна встановити з пакета AUR.


Python

Якщо у вас встановлено останню стабільну версію Python і встановлено залежності Gaphor, ви також можете встановити Gaphor за допомогою колеса з PyPI.

Якщо у вас не встановлено останню стабільну версію Python і залежностей Gaphor, дотримуйтеся розділу середовища розробки інструкції з встановлення, але не клонуйте репозиторій. За бажанням створіть віртуальне середовище. Потім виконайте наступне:

pip install gaphor
gaphor
