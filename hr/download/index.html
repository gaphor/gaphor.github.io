Preuzmi Gaphor

Postoji mnogo načina za instaliranje Gaphora. Najjednostavnije je preuzeti službeni instalacijski program za Windows ili macOS. Za Linux se Gaphor može instalirati koristeći FlatHub. Također je moguće koristiti Pythonov ugrađeni pip alat ukoliko su sve potrebne ovisnosti instalirane.

Ako si programer/ka, izvorni kod možeš pronaći na GitHubu.


Windows

Preuzmi instalacijski program

Alternativno možeš preuzeti prenosivu verziju Gaphora.

Preuzmi prenosivu verziju

Winget

Instaliraj Gaphor sa stranice winget:

winget install gaphor

Zaprikaz najnovijih napomena o izdanju i drugih detalja:

winget show gaphor

macOS

Preuzmi .dmg

Gaphor zahtijeva macOS 10.15 ili noviju verziju.

Homebrew

Instaliraj Gaphor iz jednog homebrew cask:

brew install --cask gaphor

Linux

Flatpak

Flatpak je preporučeni način za instaliranje Gaphora u Linuxu. Ako još nije postavljen, slijedi upute za instaliranje Flatpaka.

Za ručno instaliranje Gaphora:

flatpak remote-add --user --if-not-exists \
    flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install --user flathub org.gaphor.Gaphor

Arch Linux

Gaphor se može instalirati iz jednog AUR paketa.


Python

Ako imaš instaliranu najnoviju stabilnu verziju Pythona i ovisnosti za Gaphor, Gaphor možeš instalirati i pomoću PyPI.

Ako nemaš instaliranu najnoviju stabilnu verziju Pythona i ovisnosti za Gaphor, slijedi upute u odjeljku za instalaciju razvojnog okruženja, ali nemoj izraditi klon repozitorija. Opcionalno, stvori virtualno okruženje. Zatim izvrši sljedeće:

pip install gaphor
