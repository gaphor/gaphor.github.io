Gaphor herunterladen

Es gibt viele Möglichkeiten, Gaphor zu installieren. Am einfachsten ist es, den offiziellen Installer für Windows oder macOS herunterzuladen. Für Linux können Sie Gaphor via FlatHub installieren. Sie können auch das in Python eingebaute pip-Tool verwenden, solange Sie alle erforderlichen Abhängigkeiten installiert haben.

Wenn Sie ein Entwickler sind, finden Sie den Quellcode auf GitHub.


Windows

Installer heruntergeladen

Alternativ kann eine portable Version von Gaphor heruntergeladen werden.

Portable Version heruntergeladen

Winget

Installieren Sie Gaphor von winget:

winget install gaphor

Um die neuesten Versionshinweise und andere Details zu sehen:

winget show gaphor

macOS

.dmg herunterladen

Gaphor benötigt macOS 10.15 oder neuere Version.

Homebrew

Installiren Sie Gaphor von einem homebrew cask:

brew install --cask gaphor

Linux

Flatpak

Flatpak ist der empfohlene Weg, um Gaphor unter Linux zu installieren. Wenn Sie es noch nicht eingerichtet haben, folgen Sie den Anweisungen um Flatpak zu installieren.

Um Gaphor manuell zu installieren:

flatpak remote-add --user --if-not-exists \
    flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install --user flathub org.gaphor.Gaphor

Linux Architektur

Gaphor kann aus einem AUR-Paket installiert werden.


Python

Wenn Sie die neueste stabile Version von Python und die Gaphor-Abhängigkeiten installiert haben, können Sie Gaphor auch von PyPI installieren.

Wenn Sie nicht die neueste stabile Version von Python und die Gaphor-Abhängigkeiten installiert haben, folgen Sie dem Abschnitt Installationsanweisungen für die Entwicklungsumgebung, aber klonen Sie nicht das Repository. Optional können Sie eine virtuelle Umgebung erstellen. Führen Sie dann folgenden Befehl aus:

pip install gaphor
gaphor
