Contribuir

Gostaria de contribuir para o desenvolvimento do Gaphor? (Se você acha que isso é apenas algo para desenvolvedores experientes, reconsidere, há muitas maneiras de contribuir para Gaphor, não importa o seu nível de experiência ou conjunto de habilidades.

Nossa documentação técnica está hospedada no maravilhoso serviço que é Read the Docs. Não só contém detalhes necessários para fazer contribuições de código para a Gaphor, mas descreve muitas outras formas de contribuir (incluindo fazer atualizações para este site). Finalmente, ele fornece um guia para nossos processos e expectativas para os vários tipos de contribuição que você pode fazer para Gaphor.

O código-fonte está hospedado no GitHub.

Ajude a traduzir Gaphor no Weblate.

Se tiver dúvidas, basta perguntar em nossa área de discussão e alguém será capaz de apontar você na direção certa.
