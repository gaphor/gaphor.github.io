Загрузить Gaphor

Есть много способов установить Gaphor. Самый простой — загрузить официальный установщик для Windows или macOS. В Linux вы можете установить Gaphor с помощью FlatHub. Кроме того, можете использовать встроенный в Python инструмент pip, если у вас установлены все необходимые зависимости.

Если вы разработчик, то можете найти исходный код на GitHub.


Windows

Скачать установщик

Также вы можете загрузить портативную версию Gaphor.

Скачать портативную версию

Winget

Установить Gaphor используя winget:

winget install gaphor

Проверить список последних изменений:

winget show gaphor

macOS

Скачать .dmg

Gaphor требует macOS 10.15 или новее.

Homebrew

Установить Gaphor используя homebrew cask:

brew install --cask gaphor

Linux

Flatpak

Для установки Gaphor в Linux рекомендуется использовать Flatpak. Если у вас не установлен Flatpak, следуйте инструкции по установке Flatpak.

Чтобы установить Gaphor вручную:

flatpak remote-add --user --if-not-exists \
    flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install --user flathub org.gaphor.Gaphor

Arch Linux

Gaphor может быть установлен из пакета AUR.


Python

Если у вас установлена последняя стабильная версия Python и необходимые для Gaphor зависимости, то можете установить Gaphor используя wheel из PyPI.

Если у вас не установлена последняя стабильная версия Python и необходимые для Gaphor зависимости, следуйте инструкциям по установке для соответствующей среды разработки, но не клонируйте репозиторий. При желании создайте виртуальную среду. Затем выполните:

pip install gaphor
