下载Gaphor

可以采用多种方式安装Gaphor，最简单的方式是下载Windows/macOS的官方安装程序；对于Linux系统，可以使用FlatHub安装Gaphor；如果您已经安装了所有必需的依赖，您同样可以使用Python内置的pip工具进行安装。

如果您是开发者，您可以从GitHub上获得源代码。


Windows

下载安装包

您也可以下载 Gaphor 的便携版本。

下载便携式版本

Winget

通过 winget安装Gaphor：

winget install gaphor

要查看最新版本说明及其他详细信息：

winget show gaphor

macOS

下载ARM .dmg

下载Intel .dmg

Graphor 需要 macOS 10.15 或更新的版本。

Homebrew

通过 homebrew cask 获取 Gaphor：

brew install --cask gaphor

Linux

Flatpak

Flatpak是在 Linux 中安装 Gaphor 的推荐方法。如果您还没有设置好flatpak，请按照安装 Flatpak的说明进行安装。

手动安装 Gaphor：

flatpak remote-add --user --if-not-exists \
    flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install --user flathub org.gaphor.Gaphor

Arch Linux

Gaphor 可通过 Arch 软件包安装。

sudo pacman -S gaphor

Python

如果您已经安装了最新的 Python 稳定版，并安装了 Gaphor 依赖项，也可以使用 PyPI 中的wheel来安装 Gaphor。

如果您尚未安装最新的 Python 稳定版和 Gaphor 依赖项，请按照开发环境安装说明 章节操作，但不要克隆版本库。可选择创建一个 虚拟环境。 然后执行以下操作：

pip install gaphor
gaphor
