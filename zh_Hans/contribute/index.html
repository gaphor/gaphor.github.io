贡献

您是否愿意参与 Gaphor 的开发贡献？（如果您认为这仅限于资深开发者，请再考虑一下——无论您的经验水平或技术背景如何，都有多种方式可以为 Gaphor 贡献力量）。

我们的技术文档托管在优秀的Read the Docs服务平台。该文档不仅包含了为 Gaphor 进行代码贡献所需的详细信息，还概述了多种参与贡献的其他方式（包括对本网站进行更新）。最后，文档还针对不同类型的贡献，提供了我们的工作流程说明和期望标准。

源代码托管在 GitHub 平台。
